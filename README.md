# 💳 Business Card Design

Kumpulan desain **kartu nama digital** berbasis HTML dan CSS. Cocok untuk referensi desain kartu nama yang modern dan profesional.

## 🖼️ Preview

Berisi berbagai template desain kartu nama yang siap digunakan atau dimodifikasi.

## 🛠️ Tech Stack

- **HTML5** — Struktur halaman
- **CSS3** — Styling

## 🚀 Cara Melihat

Buka file `index.html` langsung di browser.
